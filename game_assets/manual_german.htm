1. Was ist Pekka Kana 2?

  
 


Pekka Kana 2 (Pekka the Rooster 2) ist ein Jump'n Run im Geiste der klassischen Plattform-Spiele wie Super Mario, Sonic der Igel, Jazz Jackrabbit, Super Frog und vieler anderer.

Pekka Kana 2 ist Freeware. Zu nichtkommerziellen Zwecken kann es von Ihnen auf jede Art und Weise weiterverbreitet werden. Dabei dürfen für eine Auslieferung von Ihnen keine höheren Kosten, als die für einen CD-Rohling erhoben werden.

3. Systemvoraussetzungen

  


- Betriebssystem: Windows 95, 98, 2000, XP*
- CPU: 500 MHz (empfohlen)
- DirectX 7.0 (mindestens)
- 16MB Speicher
- Es ist keine 3D Grafikkarte erforderlich. Allerdings sollte die Grafikkarte gute 2D Wiedergabeeigenschaften besitzen.

* Hinweis für Windows XP Nutzer: Troubleshooting Problem 1 ansehen.

5. Story

  


"Rooster Island 1"

Pekka ist ein Hahn, der auf einer Farm in einem unheimlich weit entfernten Land namens Finnland lebt. Pekka hat dort die Aufgabe, ein Hühnerhaus voller (ÜBERRASCHUNG!) Hennen zu bewachen. Alles ging seinen gewohnten Gang, bis eines Nachts...

...etwas Fürchterliches passierte: alle Hühner wurden gestohlen! Aber von wem? Nun, der Dieb war eine boshafte unehrliche Krähe, genannt "The Evil One". Ein eindeutig unheimlicher Vogel und "schlimmer Finger" obendrein. Nun denn, Pekka muß die gestohlen Hühner befreien und auf die Farm zurückbringen. Ja - und dieser häßlichen Krähe einen Tritt in den Hintern verpassen, der für einen Flug von hier bis in die Unendlichkeit reicht...

Aber das ist leichter gesagt als getan! The Evil One (ihre Freunde nennen sie kurz Evil) hat ein schreckliches Gerät, mit dem sie das Bewußtsein beinflussen kann, um ihre Armee von hypnotisierten Tieren zu kontrollieren. Diese sind zwar ebenfalls Opfer von The Evil One. Aber solange sie unter ihrer Kontrolle stehen, sind sie Pekkas (und somit auch Ihre) Gegner.

"Rooster Island 2"

Es kommt noch dicker!

Nachdem alle Hühner gerettet waren, wurde es offensichtlich, warum Pekkas Freunde in erster Linie von The Evil One "chickennapped" wurden. Aus unerfindlichen Gründen wirkt das Gehirnwäsche-Gerät von The Evil One bei den Hühnern nicht. Pekka hat außerdem herausgefunden (trotz seines winzigen Verstandes), dass The Evil One auch mit merkwürdigem Voodoozauber und Genemanipulationen herumexperimentiert hat.

Aber das ist noch nicht alles!
Glaubt man den Gerüchten, dann ist The Evil One dabei, die Entwicklung von Prototypen einer neuartigen weltbeherrschenden Waffe zu beenden: "The Metal Hen"! Gemeinsam sind diese entsetzlichen Roboter in der Lage, eine die Welt umspannende, das Bewußtsein beeinflussende Welle zu erzeugen, der auch die Hühner nicht widerstehen können.

Wieder einmal muß Pekka (mit Ihrer Hilfe) The Evil One daran hindern, die Weltherrschaft an sich zu reissen! Und wenn Sie gedacht haben, beim letzten Mal war es schon schwierig - warten Sie es ab, diesmal wird es wirklich häßlich...

Oh, bevor ich es vergesse - Pekkas Freunde sind auch wieder verschwunden.

Ist Evil nicht wirklich übel?
     
 

6. So wird gespielt

  
 

6.1 Das Ziel des Spiels

  


Das Spiel besteht aus Episoden und diese wiederum aus Leveln. Um eine Episode abzuschließen, müssen alle Level beendet werden. Das ist der Fall, wenn der jeweilige Ausgang in einem Level erreicht wird. Einige Level müssen in einer bestimmten Zeit bewältigt werden. Die Anzahl der Leben ist unbegrenzt, so dass ein Level beliebig oft wiederholt werden kann, wenn mitten im Level die Energie ausgeht und neu gestartet werden muß.

Für jedes Level wird ein eigener Hiscore und die Bestzeit notiert. Nach jedem Level werden die neuen und alten Werte miteinander verglichen und ggfs. aktualisiert. Je mehr Bonusgegenstände gesammelt und Gegner auf die Bretter geschickt werden, desto höher fällt natürlich auch die Punktzahl aus. Extrapunkte gibt es für nicht verbrauchte Energie, Gegenstände und Zeit. Die Punktzahl eines Levels wird zur Gesamtpunktzahl hinzugezählt. Sie werden neuer Episoden-Meister, wenn Sie diese Gesamtpunktzahl überbieten.

Das Spiel kann zu jeder Zeit gespeichert werden. Dabei werden allerdings nur die Gesamtpunktzahl und die erreichte Position auf der Karte festgehalten.

Das Spiel enthält zwei Episoden: "Rooster Island 1" und "Rooster Island 2".
Mögliche weitere Episoden stehen später auf der Hompage von
Piste Gamez http://pistegamez.cjb.net oder http://koti.mbnet.fi/kiwi/pistegamez
zum Download bereit.

6.3 Gegenstände

  
 


Eine kleine Auswahl der Gegenstände, die während des Spiels gesammelt werden können:

Gegenstand Name Anmerkung
 
Feder Erneuert verlorene Energie, wenn Pekka einen Treffer einstecken musste.
 
Früchte Äpfel, Organen, andere Früchte und Gemüse erhöhen die Punktzahl. Seltenere Früchte bringen natürlich auch mehr Punkte. Versuchen Sie, möglichst alle einzusammeln!
 
Megaphon Werden die Gegner mit dem Megaphon beschallt, fallen sie vor Schreck um und sind außer Gefecht gesetzt! KIKERIKI!! Die "Doodle Attacke" kann auch Gegenstände bewegen!
 
Doodle Drinks Der inoffizielle Sponsor des Spiels, Cock-A-Cola Drink Enterprises, proudly presents: Doodle Drinks ™! Doodle Drinks gibt es in vielen Geschmacksrichtungen wie Chili, Knoblauch und Eiskalter Tee. Wird ein Doodle Drink gesammelt, kann eine spezielle Doodle Attacke ausgeführt werden.
 
Easter Egg Der Legende nach macht es ein Easter egg möglich, dass auch Hähne Eier legen... Diese können dann von oben herab auf die Gegner fallengelassen werden :)
 
Schlüssel Schlüssel öffnen Schlösser, logisch. Werden alle Schlüssel in einem Level gesammelt, so öffnen sich auch sämtliche Schlösser. Ein Schloß kann ebenfalls geöffnet werden, wenn es mit einem Schlüssel berührt wird.
 
Geschenk Kleine Geschenke erhalten die Freundschaft. Sie enthalten Gegenstände wie z.b. eine Feder. Diese Geschenke werden im Inventar abgelegt. Das Inventar kann bis zu sechs Gegenstände aufnehmen. Ein Gegenstand kann durch einen Druck auf die Leertaste (oder der von Ihnen dafür vorgesehenen Taste) benutzt werden.
 
Schachtel Schachteln enthalten entweder Bonusgegenstände oder manchmal eingesperrte Tiere. Also: aufmachen und sich überraschen lassen!
 
Uhr Mehr Zeit gefällig? Dann sammeln sie eine Uhr!
 
Mysteriöser Trank Ich würde auch gern wissen, was es damit auf sich hat.
 
Info Tafel Wird sie berührt, erscheint ein ein kleines Fenster mit einer informativen Meldung.

Tipp: Versuchen Sie, Bonusgenstände (wie etwa verfaulte Äpfel) auf die Gegner zu werfen ;)

  
     
 

6.4 Die Darsteller (Who's Who in diesem Drama)

  
 


Während Ihres Abenteuers werden Ihnen viele verschiedene Charaktere begegnen. Die meisten davon sind Gegner, aber eben nicht alle.

Charakter Name Anmerkung
 
Pekka Pekka ist ein (Hühner-) Vogel mit dem Verstand einer Erdnuss. Er selbst würde nie im Leben die verschwundenen Hennen finden. Also müssen Sie ihm dabei helfen und zeigen, wo es lang geht. Versuchen Sie, ihn am Leben zu lassen, okay? Oh, wussten Sie eigentlich, dass Pekka noch Brüder hat? Nun, es könnte sein, dass Ihnen diese auf Ihrer Reise begegnen.
 
Huhn Retten Sie Pekkas Freunde, bevor The Evil One
Chicken McNuggets aus ihnen machen kann! Obgleich die Hühnerschar bis zum Ende der Episode zusammen gehalten wird, können Ihnen unterwegs andere Hühner begegnen. Lassen Sie sich nicht beirren und behalten Sie Ihr Ziel im Auge!
 
Küken Die Küken werden versuchen, Ihnen zu folgen. Da sie klein sind, erreichen sie gut enge Örtlichkeiten und können in manchen Situationen sehr hilfreich sein.
 
Frosch Igitt! Grün und glitschig! Entweder ausweichen oder aufblasen :)
Liebe Kinder! Natürlich blasen wir keine Frösche auf! Das Megaphon tut's auch!
 
Igel Ein Igel auf einem Skateboard ist eine schlechte Nachricht! Und eine noch schlechtere Nachricht: einige von ihnen tragen auch noch Kopfhörer, so dass ihnen das Megaphon nichts anhaben kann.
 
Boxender Hase Er wird versuchen Sie zu jagen, sobald er sie erblickt! Versuchen Sie, sich hinter ihn zu schleichen, während er in die andere Richtung schaut und verpassen Sie ihm dann eine von dort aus!
 
Bienenstock Der Stock selbst ist naürlich nicht gefährlich. Sollten Sie allerdings anfangen, die Bienen darin zu ärgern ... Tipp: Bienen sind keine Weltmeister im Schwimmen.
 
Chili Schwein Kommen Sie den Chili Schweinen und ihrem feurigen Atem nicht zu nahe, wenn Sie Pekka nicht den Bürzel ansengen wollen!

Dieses sind nur einige der Charaktere, die Ihnen während Ihrer Reise begegnen werden.

  
 

6.5 Besondere Objekte

  
 


Objekt Name Anmerkung
 
Start Die Startmarke kennzeichnet den Startpunkt eines Levels. In einem Level kann es mehrere Startpunkte geben...
 
Ausgang Diese Marke zu erreichen, ist Ihr Ziel in jedem Level.
 
Schloß Schlösser können auf zwei Arten geöffnet werden. Haben Sie die erforderliche Anzahl von Schlüsseln gesammelt, öffnen sich alle Schlösser. Schlösser öffnen sich auch, wenn sie von einem Schlüssel berührt werden. Diese Schlüssel sehen anders aus...
 
Grüne Schalter Ein Sprung auf den Schalter und alle Wände mit grünen Pfeilen werden bewegt. Aber Beeilung! Die Wände kehren nach einer gewissen Zeit in ihrer Ausgangspositionen zurück!
 
Blaue Schalter Funktionieren wie die grünen Schalter. Sie haben allerdings nur Auswirkung auf die blau markierten Wände.
 
Orange Schalter Funktionieren wie die grünen Schalter, jedoch verschieben Orange Schalter keine Wände, sondern schalten Feuerfallen aus. Und nochmals Beeilung, denn dieses geschieht auch nur für eine kurze Zeit!
 
Bewegliche Wände Sie werden verschoben, wenn der entsprechende Schalter aktiviert wurde.
 
Fallen Fallen werden durch solche Steine gekennzeichnet und können ein- bzw. ausgeschaltet werden. Finden Sie heraus, wie das geht... ;)

Schalter können auch aktiviert werden, wenn Sie etwas auf sie herabfallen lassen, das schwer genug ist.
Werden ausreichend leichtere Sachen auf ihnen gestapelt, funktioniert das auch.
In einigen Leveln werden Fallen durch Schalter ausgelöst. Seien Sie also vorsichtig!

  
     
 

7. Troubleshooting

  


Problem 1: Keine korrekte Grafikdarstellung und unerwartete Programmabstürze
Lösung: Als Windows XP Nutzer sollten Sie eine Bildschirmauflösung von 640x480 ausprobieren. Dieses kann bei anderen Betriebssystem mit ähnlichen Problemen auch für Abhilfe sorgen.

Problem 2: Das Spiel startet nicht!
Lösung: Prüfen Sie, ob Ihr PC die Systemvoraussetzungen erfüllt. Falls JA.. umm... könnte das Archiv beschädigt sein und Sie sollten die neueste Version von der Piste Gamez Homepage herunterladen.

Problem 3: Das Spiel läuft laaaaaaaangsam!
Lösung: Schalten Sie im Grafikmenü Transparent FX ab. Das macht eine Menge aus. Besteht das Problem weiterhin, wählen sie Double Speed im selben Menü, allerdings auf eigene Gefahr :) Außerdem empfehle ich, alle anderen Programme während des Spielens zu beenden.

Problem 4: Das Spiel ist immer noch sehr laaaaaaaangsam!
Lösung: PK2 ist für eine Darstellung von 85 Frames pro Sekunde ausgelegt. Ist die Bildwiederholungsrate ihres Monitors geringer als 85 MHz wird das Spiel dadurch ausgebremst. Erhöhen Sie also die Bildwiederholungsrate des Monitors. Wenn Sie während des Spiels die Taste Einfg drücken, wird Ihnen der aktuelle Wert angezeigt.
